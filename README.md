# Online Retail Sales & Customer Insights Dashboard

## Overview
This project presents an interactive Power BI dashboard built to analyze online retail
sales performance and customer behavior.

## Tools Used
- Power BI
- DAX
- Excel / CSV

## Dashboard Features
- Total revenue and quantity sold analysis
- Country-wise sales performance
- Monthly and yearly sales trends
- RFM-based customer segmentation
- Interactive slicers for dynamic analysis

## Key Insights
- Identified top revenue-generating countries
- Observed seasonal sales patterns
- Found that a small group of customers contributes significantly to revenue
